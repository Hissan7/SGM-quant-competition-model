# Strand Global Macro Quant Competition 

First README of project 

ML-driven, macro-aware portfolio optimization framework.

## Core ideas
- Sell-side target–based return modeling
- Uncertainty-aware robust optimization
- Explicit macro regime rotation (defensive → growth)
- Strict risk, turnover, and position constraints
